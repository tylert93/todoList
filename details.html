Overview

This Todo-List provides the user with three basic functions; adding new items to the list, deleting items form the list and checking of items to show them have been completed. It is not a fully developed application and is more of a concept project. It doesn’t have any database functionality and so whenever the page is reloaded, the list items reset back to their default values and all data entered by the user is lost.
The applications functions were built using jQuery. The ‘.keypress()’ method is used to detect whenever the user presses ‘enter’. When this happens the ‘.val()’ method is used to get whatever text has been entered into the input field and add it to the list using ‘.append()’. If the user clicks on a list item, the ‘.toggleClass()’ to used to add or remove the ‘completed’ class. This makes an item greyed out, puts a line through it and makes a green tick appear. Whenever a trash can icon is clicked a combination of ‘.parent().’, ‘.fadeOut()’ and ‘.remove()’ are used to delete the item which the icon belonged to. ‘event.stopPropagation()’ is used to prevent events firing on parent elements that were also clicked and therefore stops the list item becoming ‘checked’.
This project certainly has potentially to become much more than it already is and is a project I plan to return to once I have developed the skills necessary ‘todo’ so. Pardon the pun.
